# 🧠 DIU WISE: AI-Powered Student Wellness & Success Hub

[![Python Version](https://img.shields.io/badge/python-3.10%2B-blue)](https://www.python.org/)
[![Flask](https://img.shields.io/badge/flask-v3.0-green)](https://flask.palletsprojects.com/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**DIU WISE** is an advanced, human-centric ecosystem designed to foster student wellbeing and academic excellence through artificial intelligence. By identifying silent signals of distress and providing proactive, non-clinical support, DIU WISE bridges the gap between academic pressure and mental health.

---

## ✨ Core Features

### 🌈 Digital Wellness Twin (Aura)
Visualize your emotional state through a dynamic "Aura" that adapts based on your recent mood, stress levels, and academic engagement.

### 🤖 Floating AI Wellness Assistant
A cute, draggable, and interactive AI guide that follows you across the platform, providing instant supportive perspective and academic tips.

### 📊 Student Success Hub
A comprehensive profile section tracking:
- **Academic Metrics**: GPA, credit load, and submission status.
- **Wellbeing Trends**: Mood history, stress indicators, and sleep quality analysis.
- **Goals & Achievements**: Track personal study goals and earn badges for consistency.

### 📈 Predictive Analytics & Triage
- **Early Distress Detection**: Analyzes attendance and submission patterns to flag students needing support.
- **Emotion Analysis**: Real-time supportive feedback based on journal entries and reflections.
- **Actionable Guidance**: Personalized "Wellbeing Plans" generated by AI.

### 🆘 Crisis Safety Net
When the assistant detects high distress in a conversation, it doesn't just label the risk — it acts. A calming overlay launches an interactive box-breathing exercise and surfaces verified Bangladesh emergency helplines (National Emergency **999**, **Kaan Pete Roi**, **Moner Bondhu**) with one-tap calling, closing the loop between *detection* and *real support*.

### 🔒 Privacy & Data Sovereignty
Granular controls allow students to manage data collection, download their history, or exercise their "right to be forgotten."

## 🚀 Getting Started

### Prerequisites
- Python 3.10 or higher
- Git

### Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/foysalpranto121/DIU-WISE-AI.git
   cd DIU-WISE-AI
   ```

2. **Setup the Backend**
   ```bash
   cd backend
   python -m venv .venv
   source .venv/bin/activate  # Windows: .venv\Scripts\activate
   pip install -r requirements.txt
   ```

3. **Configure Environment**
   Create a `.env` file in the `backend` directory:
   ```env
   SECRET_KEY=your_secret_key
   DATABASE_URL=sqlite:///wellness.db  # Or your PostgreSQL URL
   ```

4. **Run the Application**
   ```bash
   python app.py
   ```
   Access the platform at `http://127.0.0.1:5000`.
